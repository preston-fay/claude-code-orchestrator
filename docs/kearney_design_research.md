# Kearney Brand Design Research

**Generated:** 2025-11-19
**Purpose:** Document Kearney brand design principles for KDS alignment

## Brand Guidelines Summary

Based on Kearney brand materials and visual identity:

### Core Visual Principle

**"White + Slate + Purple Accent"**

Kearney's visual language is built on a foundation of white and neutral slate/greys, with purple used **sparingly** as an accent color for emphasis and key information. This creates a clean, high-contrast, editorial aesthetic.

---

## Color Usage Patterns

### Background & Surfaces
- **Primary background:** White (`#FFFFFF`)
- **Secondary surfaces:** Light grey (`#F5F5F5`, `#E6E6E6`)
- **Elevated surfaces:** White with subtle shadows
- **Dark mode (if used):** Very dark grey or black (`#000000`, `#1E1E1E`)

### Text
- **Primary text:** Dark charcoal (`#1E1E1E`)
- **Secondary/muted text:** Mid grey (`#787878`, `#A5A5A5`)
- **On dark backgrounds:** White (`#FFFFFF`)

### Emphasis & Accents
- **Primary emphasis:** Kearney Purple (`#7823DC`)
- **Hover states:** Lighter purple (`#9150E1`)
- **Links:** Purple (`#7823DC`)
- **CTAs (primary buttons):** Purple background, white text
- **Highlights:** Light purple tint (`#E6D2FA`, `#F4EDFF`)

### Borders & Dividers
- **Subtle borders:** Light grey (`#D2D2D2`, `#E6E6E6`)
- **Medium borders:** Mid grey (`#C8C8C8`)
- **Focused/active:** Purple (`#7823DC`)

### NO Color-Coded Semantics
- **NO green for success**
- **NO red for errors**
- **NO orange for warnings**
- **NO blue** (old brand color - deprecated)

Instead: Use **text labels, symbols, or purple intensity** to convey state.

## Typography Patterns

### Font Family
- **Headline:** Inter (bold, semi-bold)
- **Body:** Inter (regular, medium)
- **Fallback:** Arial, system fonts
- **Monospace:** SF Mono, Roboto Mono (for code)

### Font Weights
- **Headlines:** 600-700 (Semi-bold to Bold)
- **Subheadings:** 500-600 (Medium to Semi-bold)
- **Body:** 400 (Regular)
- **Emphasis:** 500 (Medium)
- **Muted text:** 400 (Regular)

### Casing Rules
- **NO ALL CAPS for headings** (critical rule)
- Use **sentence case** or **title case** for headings
- Use **sentence case** for body text, labels, CTAs

Examples:
- ❌ `ALL CAPS HEADLINE`
- ✅ `Sentence case headline`
- ✅ `Title Case Headline`

### Size Scale
- **4xl:** 3.815rem (60px+) - Hero/marketing
- **3xl:** 3.052rem (48px) - Page titles
- **2xl:** 2.441rem (39px) - Section titles
- **xl:** 1.953rem (31px) - Card titles
- **lg:** 1.563rem (25px) - Subheadings
- **md:** 1.25rem (20px) - Emphasized body
- **base:** 1rem (16px) - Body text
- **sm:** 0.8rem (13px) - Captions, labels
- **xs:** 0.64rem (10px) - Fine print

### Line Height
- **Headlines:** 1.2 (tight)
- **Body:** 1.5 (normal)
- **Captions:** 1.5-1.75 (relaxed)

### Letter Spacing
- **Headlines:** -0.01em to -0.02em (tight)
- **Body:** 0 (normal)
- **Small caps/labels:** 0.02em (wide) - if used

## Layout & Spacing Patterns

### Whitespace Philosophy
**"Generous whitespace" is a brand requirement.**

- Use ample padding and margins
- Don't cram content
- Let elements breathe
- Think "NYT editorial" - clean, airy, uncluttered

### Grid & Structure
- **Cards:** Square or rectangular (no pills), subtle border or shadow
- **Border radius:** Small (2-4px), never fully rounded except for pills/badges
- **Grid gaps:** 16-24px minimum between cards/sections
- **Container padding:** 24-32px (desktop), 16-20px (mobile)

### Component Patterns

#### Cards
- Background: White or light grey (`#F5F5F5`)
- Border: 1px solid light grey (`#E0E0E0` → use `#D2D2D2` or `#E6E6E6`)
- Border radius: 2-4px (subtle)
- Shadow: Minimal (1-2px offset) or none
- Padding: 24-32px
- **NO gradient backgrounds**

#### Buttons
- **Primary:** Purple background (`#7823DC`), white text, 2-4px radius
  - Hover: Lighter purple (`#9150E1`)
- **Secondary:** Transparent/white background, grey border (`#C8C8C8`), dark text (`#1E1E1E`)
  - Hover: Light grey background (`#F5F5F5`)
- **Text only:** Purple text (`#7823DC`), no background
  - Hover: Underline or darker purple

#### Links
- Color: Purple (`#7823DC`)
- Underline: Optional (on hover)
- Visited: Same purple (don't use different color)

#### KPI Cards
- Background: White or `#F5F5F5`
- Large number: Dark (`#1E1E1E`) or purple (`#7823DC`) for emphasis
- Label: Grey (`#787878`), sentence case, no ALL CAPS
- Border: Subtle left accent (4px purple) or full border (`#D2D2D2`)
- **NO colored backgrounds for trend indicators** - use neutral + text

#### Tables
- Header: Light grey background (`#F5F5F5`), dark text (`#1E1E1E`), bold
- Rows: White / very light grey alternating (`#FFFFFF` / `#FAFAFA`)
- Borders: Light grey (`#E6E6E6`, `#D2D2D2`)
- Hover: Light grey row background (`#F5F5F5`)
- **NO gradient zebra stripes**

#### Badges/Tags
- Background: Light purple (`#E6D2FA`, `#F4EDFF`) or light grey (`#F5F5F5`)
- Text: Purple (`#7823DC`) or dark grey (`#4B4B4B`)
- Border: Optional (1px same as text color, subtle)
- Border radius: 2-4px (small capsule)
- Padding: 4px 8px (compact)

## Chart & Data Viz Patterns

### Color Palette for Charts
**Purple-first, grey-secondary:**

1. **Single series:** Use primary purple (`#7823DC`)
2. **Multi-series:** Use purple gradient:
   - Series 1: `#7823DC` (primary)
   - Series 2: `#9150E1`
   - Series 3: `#AF7DEB`
   - Series 4: `#C8A5F0`
   - Series 5: `#E6D2FA`
   - Series 6+: Add greys (`#A5A5A5`, `#787878`, `#D2D2D2`)

3. **Comparison/diverging:** Purple vs grey:
   - Positive/primary: Purple shades
   - Negative/secondary: Grey shades

### Chart Elements
- **Grid lines:** NONE (critical brand requirement)
- **Axes:** Thin grey lines (`#D2D2D2`)
- **Axis labels:** Muted grey text (`#787878`)
- **Data labels:** Dark text (`#1E1E1E`) or on-data white text
- **Legend:** Below chart, purple bullets, grey text
- **Tooltips:** White/light grey background, dark text, subtle border

### Chart Types
- **Line charts:** 2px line width, slight curve (tension 0.3), no area fill unless stacked
- **Bar charts:** 4px border radius, no outlines, minimal gap
- **Pie/Doughnut:** 60% cutout, no gradient, purple → grey color order
- **Scatter:** 4-6px radius, purple fill, no stroke

## Visual Mood & Principles

### Editorial Aesthetic
- Think **New York Times, Financial Times, Economist**
- Clean, sophisticated, data-driven
- High contrast (dark text on white)
- Generous whitespace
- Minimal ornamentation

### Brand Personality
- **Professional, not playful** (no emojis, no rounded UI)
- **Confident, not showy** (no gradients, no shadows unless subtle)
- **Clear, not cluttered** (generous spacing, simple layouts)
- **Data-driven, not decorative** (charts emphasize data, not design)

### Design Principles
1. **White + slate foundation:** Start with white/grey base
2. **Purple for emphasis only:** Use purple sparingly for key actions/info
3. **High contrast:** Dark text on light backgrounds (or vice versa for dark mode)
4. **Generous whitespace:** Don't cram, let content breathe
5. **NO gridlines:** Charts must have no gridlines (brand requirement)
6. **Label-first:** Put labels directly on data marks when possible
7. **Sentence case:** No ALL CAPS headings
8. **Minimal ornament:** No gradients, no heavy shadows, no decorative elements

---

## Specific Rules (Violations to Avoid)

### ❌ Forbidden
- ALL CAPS headings
- Green/red/orange semantic colors
- Blue (old brand color)
- Gradient backgrounds
- Chart gridlines
- Heavy drop shadows
- Rounded/pill UI elements (except small badges)
- Emojis in professional content
- Comic/playful fonts

### ✅ Required
- Sentence or title case headings
- Purple + grey color palette only
- Clean, flat backgrounds (white or light grey)
- Generous whitespace
- Inter font family
- High contrast text
- Minimal border radius (2-4px)
- Subtle shadows (1-2px) or none
- Professional, editorial tone

## Application to KDS Components

### KPI Cards
- **Background:** `#F5F5F5` (light grey) or `#FFFFFF` (white)

- **Padding:** 24px

  - Background: `#F5F5F5`
  - Text: `#1E1E1E`, bold (600-700)

  - Background: Alternating `#FFFFFF` / `#FAFAFA`
  - Text: `#1E1E1E`
  - Padding: 12-16px
  - Hover: `#F5F5F5`
- **Borders:** 1px `#E6E6E6` between rows

- **Background:** `#E6D2FA` or `#F4EDFF` (light purple)
- **Text:** `#7823DC` (purple)
- **Border:** Optional 1px `#C8A5F0`
- **Border radius:** 4px
- **Padding:** 4px 12px
- **Font:** sm size, medium weight (500)

### Buttons
- **Primary:**
  - Background: `#7823DC`

  - Hover: `#9150E1`
  - Border radius: 4px
  - Padding: 10px 20px

  - Background: `#FFFFFF`

  - Border: 1px `#C8C8C8`
