# Kearney Data Platform

The Kearney Data Platform is a comprehensive, enterprise-grade solution for data analytics, visualization, and model registry management. Built with brand-locked design principles and production-ready infrastructure.

## Platform Overview

The platform consists of:

- **FastAPI Server** - High-performance REST API with OpenAPI documentation
- **Python CLI** - Powerful command-line orchestrator with Typer
- **React Frontend** - Modern web application with Kearney design system
- **Model Registry** - MLOps-ready model versioning and metadata
- **Security Layer** - Enterprise authentication, authorization, and audit logging
- **Design System** - Brand-locked components with no emojis, no gridlines

## Brand Principles

The platform adheres to strict Kearney brand guidelines:

- **Typography**: Inter font with Arial fallback
- **No Emojis**: Professional, text-only communication
- **No Gridlines**: Clean, uncluttered visualizations
- **Label-First**: Data labels prioritized over decorative elements
- **Spot Color**: Strategic use of Kearney purple for emphasis

## Architecture

The platform is built on a modern, cloud-native architecture:

- **Backend**: FastAPI + DuckDB for high-performance analytics
- **Frontend**: React + TypeScript + Vite
- **Deployment**: AWS Amplify + Lambda + S3
- **CI/CD**: GitHub Actions with automated testing and deployment
- **Documentation**: Docusaurus with auto-generated API and CLI docs
